# 🐾 Cat vs Dog Image Classifier

This web app classifies uploaded images as **Cat** or **Dog** using a pre-trained MobileNetV2 model.

## 🚀 Try it live
👉 [Streamlit App](https://cat-dog-classification-yuvraj.streamlit.app/)

## 🧠 Model
Pretrained using MobileNetV2 on Kaggle Dogs vs Cats dataset.

## 📁 Note
Dataset and large models are not included in the repo due to GitHub size limits.

## 🛠️ Tech Stack
- TensorFlow + Keras
- Streamlit
- MobileNetV2 (Transfer Learning)
